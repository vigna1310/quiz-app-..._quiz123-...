Here's a complete, simple quiz web app that you can deploy directly on GitHub Pages. It features a question "What is 2 + 2?", multiple choice options, and reveals whether the selected answer is correct or not. Just copy the following code into a file named `index.html` and push it to your GitHub repository's root. Your GitHub Pages site will serve this page. ```html Simple Math Quiz

Simple Math Quiz

What is 2 + 2?
``` ### How it works: - Presents a question and four answer options. - When a user clicks an option, it displays whether they are correct or wrong. - The feedback text appears below the options with color cues. ### Deployment: - Save this code as `index.html` in your project repository. - Push to GitHub. - Enable GitHub Pages in your repository settings (usually from the "Pages" tab). - Your quiz will be live at `https://.github.io//`. Feel free to customize the question, options, styling, or add more questions!
